# exchange_table
> exchange table with AngularJS

## Usage

```
$ git clone git@github.com:jodeng110/exchange_table.git <my-new-project>
```

```
$ cd <my-new-project>
```

### Install

```
$ npm install -g gulp bower bower-installer
```

```
$ npm install
```

```
$ bower install
```

```
$ bower-installer
```

### Build
```
$ gulp build
```

### Run development mode

```
$ gulp develop
```

open Web Browser : http://localhost:3333/
